Trivia

Start a quiz. The category parameter specifies the quiz's subject (History, Art, etc.) and length parameter is the length of the quiz, between 3 and 15 questions. Quiz is multiple-choice and others can join the quiz.
Example:
?quiz general-knowledge 5
Aliases:
q, qu, qm, quizme


Start a trivia quiz that is a certain length that is between 5 and 50 questions (length parameter). The trivia questions are answered by entering the answer into the channel. Other people can join this quiz.
Example:
?trivia 5
Aliases
t, tr, triv


See a list of categories for ?quiz.

?categories

Custom Quizzes

Run your custom quiz for you and others to play, if you have created a quiz. The custom quiz is just like the ?quiz quizzes.
Example:
?custom
Aliases
cus


Sends your custom quiz (if one has been made) in an embed for you to see.
Example:
?show
Aliases
sh


Add a question to your custom quiz. All parameters must be in "". The parameters are:
Question: The question that will be shown (remember the custom quizzes are in multiple choice format)
Answer: The correct answer option.
Incorrect options: The rest of the other options that are incorrect. There can be no more than 3 incorrect options.
Example:
?add "Is Quix a cool bot?" "yes" "no" "i want a hamburger"
Aliases
None


Deletes a question based on it's index (e.g. Question 1, Question 5, etc.)
Example:
?delete 2
Aliases
d


Completely wipes your custom quiz. Quiz data is un-recoverable.
Example:
?reset
Aliases
None


Purchasing

Shows the Quix shop of upgrades, name used in ?purchase, and cost in tokens (currency).
Example:
?shop
Aliases
sh


Purchase an upgrade for your profile. Parameter upgrade name is what you want to change ("color" (Profile2 color) or "join" (Join message)), and parameter new is the new value of the upgrade (like "I have joined the Quiz" for upgrade "join")
Example:
?purchase color 255 255 255
?purchase join I've joined the quiz.
Aliases
None


Statistics

See the profile of you or another user. If user parameter is ommitted, it shows your profile. Profile is in an image and does not show profile color (see ?purchase).
Example:
?profile
?profile @nyos.exe
Aliases
prof, p
